# Graphics Inventory

All graphics live in `/public/graphics`.

## Files
- `grid.svg`: subtle background grid used by hero sections.
- `wave.svg`: decorative wave accent (optional secondary use).
- `droplet.svg`: water droplet illustration (optional secondary use).
- `og-default.png`: default OpenGraph image placeholder (solid brand tint).

## Usage
- Background grid: applied via `.bg-grid` utility in `app/globals.css`.
- Icons or accents can be added as inline `<img>` or `next/image` when needed.

## Notes
- Keep SVGs lightweight and monochrome where possible.
- Update this file when adding or replacing graphics.
